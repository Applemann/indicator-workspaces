Elementary OS Workspace Indicator.
Configure workspaces and interact with them in the panel.

Indicator Workspaces is open source. It's hosted, developed, and maintained on GitHub.

Installing

Indicator Workspaces is distributed as a Debain package file (.deb). The latest stable release can always be downloaded from this page.

To install either open the downloaded file, or navigate to the file in the terminal and run:

$ sudo debpkg -i indicator-workspaces_0.1_all.deb

Once installed, you will need to log out and back in for the indicator to appear.

You can also view the source code and install it manually via GitHub.

Upgrading

Downloading and installing the latest verion will overwrite the currently installed version. You will need to log out and back in for the new indicator to run.

Using & Configuring

Indicator Workspaces works with both dynamic workspaces and well as a fixed number of workspaces.

Workspaces from 1 - 9 can be named manually through the preferences. If a workspace doesn't have a name, it is given one in the form of 'Workspace (Number)'. Changes to the workspaces are applied immediately.
